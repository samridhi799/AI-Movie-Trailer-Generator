# 🎬 AI Movie Trailer Generator

An AI-powered movie trailer generator that creates cinematic trailers using Google Gemini, Edge-TTS, and MoviePy.

## Features

- AI Story Generation
- AI Image Generation
- AI Voice-over
- Cinematic Zoom Effects
- Automatic Trailer Creation

## Technologies Used

- Python
- Google Gemini
- Edge-TTS
- MoviePy
- Pillow

## Installation

```bash
pip install -r requirements.txt
```

## Run

Open the notebook and run all cells.

## Output

The project generates:
- AI-generated images
- AI narration
- Cinematic movie trailer
